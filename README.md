# Student Data Analysis Project

## Project Description
This project analyzes student data using Python and Pandas.

## Project Features
- Load Data
- Inspect Data
- Clean Data
- Transform Data
- Filter Data
- Analyze Data
- Sorting
- Grouping
- Statistics
- Report Generation
- Export Data

## Technologies Used
- Python
- Pandas
- CSV

## How to Run the Project
1. Open the project in VS Code.
2. Open the Terminal.
3. Run the following command:

python main.py 

## Conclusion 
This project demonstrates how Python and the Pandas library can be used to load, clean, transform, filter, analyze, and export student data. It helps in understanding real-world data analysis techniques and improves practical knowledge of data processing using Python.
